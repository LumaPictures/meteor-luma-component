## Local Example
```
$ git clone https://github.com/LumaPictures/meteor-luma-component
$ cd LumaPictures/meteor-luma-component
$ mrt && meteor
```

## Local Tests
```
$ git clone https://github.com/LumaPictures/meteor-luma-component

$ mrt && meteor test-packages luma-component
```
